About

This game was created by two cool guys Joey and Dennis as a project of the lecture Web Technologies

The goal of this lecture was to create a web application based on a game implemented in the third semester. The old project is continued and a WebUI has to be programmed.
This Poker game includes following web technologies:

  • HTML
  • CSS
  • Bootstrap
  • Javascript
  • jQuery
  • AJAX
  • WebSockets
  • AngularJS
  • Polymer
  • Deployment with heroku
  • Authentication with github
}
